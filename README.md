# py-jpgtotext
Just little funny project
## Setup requirements:
**Python 3.+**
```bash
sudo pip install Image Colr numpy
```
**Or**
```bash
sudo python -m pip install Image Colr numpy
```
## Usage:
```bash
python main.py [--cols=<width>] <image.jpg>
```
